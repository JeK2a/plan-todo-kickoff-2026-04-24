## Статус задачи #3

Реализован первый сквозной сценарий MVP в веб-интерфейсе:

- Добавление новой задачи.
- Фильтрация по статусам: все, активные, выполненные.
- Переключение статуса задачи (готово/вернуть).
- Удаление задачи и очистка выполненных.
- Сохранение состояния в `localStorage`.

### Локальный запуск

1. Откройте папку проекта в терминале.
2. Выполните:
   - `python3 -m http.server 4173`
3. Откройте [http://localhost:4173](http://localhost:4173).
