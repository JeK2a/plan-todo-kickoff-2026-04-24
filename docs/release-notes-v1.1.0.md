# Release Notes v1.1.0

Дата релиза: 2026-04-24

## Что вошло в релиз

- Добавлено inline-редактирование задач:
  - вход в режим редактирования,
  - сохранение по кнопке и Enter,
  - отмена по кнопке и Escape,
  - защита от сохранения пустого значения.
- Добавлены дедлайны задач и логика определения просрочки по локальной дате.
- Добавлена визуальная метка просроченных активных задач.
- Добавлены импорт и экспорт задач в JSON:
  - экспорт в файл,
  - импорт с валидацией формата,
  - сообщения об ошибках в UI.
- Добавлен UI smoke-тест основного сценария на Playwright.
- CI расширен: запуск unit + smoke тестов в Node 20.
- Добавлена Docker-конфигурация для локального запуска (`Dockerfile`).

## Технический срез

- UI и сценарии: `index.html`, `app.js`, `styles.css`.
- Логика и тесты: `taskLogic.js`, `taskLogic.test.js`.
- E2E smoke: `playwright.config.js`, `tests/smoke.spec.js`.
- CI: `.github/workflows/ci.yml`.
- Контейнеризация: `Dockerfile`, `.dockerignore`.

## Статус задач

Все задачи итерации `v1.1.0` (`#6`, `#7`, `#8`, `#9`, `#10`) завершены и переведены в `Done`.

## Известные нюансы

- Для локального запуска smoke-тестов нужен Node.js 18+.
- В CI используется Node.js 20, поэтому smoke-тесты выполняются в поддерживаемой среде.
