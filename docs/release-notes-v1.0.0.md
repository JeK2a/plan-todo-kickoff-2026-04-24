# Release Notes v1.0.0

Дата релиза: 2026-04-24

## Что вошло в релиз

- Реализовано MVP веб-приложение TODO с полным пользовательским сценарием:
  - создание задачи,
  - изменение статуса,
  - удаление задачи,
  - фильтрация по статусам,
  - очистка выполненных.
- Добавлено сохранение состояния задач в `localStorage`.
- Вынесена бизнес-логика задач в отдельный модуль `taskLogic.js`.
- Добавлены unit-тесты на критичную логику (`node:test`).
- Настроен CI workflow в GitHub Actions (проверки + запуск тестов).
- Документированы scope/non-goals и acceptance criteria для `v1.0.0`.

## Технический срез

- Основные файлы интерфейса: `index.html`, `styles.css`, `app.js`.
- Логика и тесты: `taskLogic.js`, `taskLogic.test.js`.
- CI: `.github/workflows/ci.yml`.
- Документация: `docs/mvp-scope.md`, `docs/acceptance-criteria-v1.md`.

## Статус задач

Все задачи стартового плана (`#1`, `#2`, `#3`, `#4`, `#5`) завершены и переведены в `Done`.

## Известные ограничения

- Данные хранятся только локально в браузере (без серверного API и БД).
- Нет авторизации и многопользовательского режима.
